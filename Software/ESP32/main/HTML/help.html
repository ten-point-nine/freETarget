Welcome to the FreeETarget help page.

Available target commands.

<target-name>

This help page

<target-name>/target

Launch the target page.  Runs until all of the shots have been displayed.  Use REFRESH to obtain the next shots

<target-name>/target?auto

Launch the target page.   New shots will automatically appear.  Use <target-name>:81/target?stop to end the session

<target-name>/target?stop

Stops a running session started with <target-name>/target?auto

<target-name>/who

Obtain the target configuration
