The web based FreeETarget interface is in two parts:

1 – The main target page showing your shots, and;

2 – A control page to manage the target remotely.

 

You will need to know the name or IP address of your target.

 

To start the target, enter the target URL;

<target_name>.local

This starts the target and records your shots. The target will automatically disconnect after 15 minutes of inactivity.

 

To control the target, access the menu port

<target_name>.local:8080

This will present the menu buttons:

START – Clear any shots and start recording

STOP – Stop recording shots and return to the HELP page

 

To find out information about your target, use the WHO command

<target-name>.local/who

Or

<target-name>.local:8080/who
